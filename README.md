<h1>Nikafeis - frontend</h1>

<h2>Nikafeis is a personal project, the objective is show all my skills and knowledge in a simple way.</h2>

<h2>The project is developed in </h2>

<h4>React.js</h4>
<p>A JavaScript library for building user interfaces.</p>

<h4>MUI Core</h4>
<p>Material-UI is a collection of components built to work together.</p>

<h4>Typescript</h4>
<p>TypeScript is a typed superset of JavaScript that compiles to plain JavaScript.</p>

<h2>Installation</h2>
<p>To install the project, you need to install the dependencies:</p>

<code>git clone https://github.com/JsMarenco/nikafeis.git</code>
<code>cd nikafeis</code>
<code>npm install</code>
<code>npm start</code>
